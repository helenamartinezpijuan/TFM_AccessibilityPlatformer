# README FILE

## Comandos
A continuación se describen los comandos necesarios para interactuar con el juego. Se ha utilizado el New System Input de Unity.

### Move
Solo se permite el movimiento en horizontal. Los comandos según el controller utilizado son los siguientes:
- *Teclado* - Flechas izquierda/derecha () y teclas A y D (del clásico WASD).
- *Pointers* - Touch and Mouse support.
- *Gamepad* - x
- *Joystick* - x
- *HID* - x ?
